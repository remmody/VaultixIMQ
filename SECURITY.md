# Security Policy

## Supported Versions

Currently supported versions for security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of VaultixIMQ seriously. If you believe you have found a security vulnerability, please report it to us as follows:

1.  **Use GitHub Private Vulnerability Reporting**: Go to the "Security" tab of this repository and select "Vulnerability reporting" to submit a confidential report.
2.  **Alternatively**, if private reporting is unavailable, please open a GitHub Issue with the prefix `[SECURITY]` but avoid sharing sensitive exploit details publicly until we can coordinate a fix.

We will acknowledge your report within 48 hours and provide a timeline for a fix if the vulnerability is confirmed.
