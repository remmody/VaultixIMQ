# VaultixIMQ

![GitHub Release](https://img.shields.io/github/v/release/remmody/VaultixIMQ)
![GitHub Downloads](https://img.shields.io/github/downloads/remmody/VaultixIMQ/total)

Надежный и современный IMAP-клиент, разработанный для управления большим количеством почтовых ящиков.
 подключенных почтовых ящиков.
 Программа обеспечивает безопасную среду для синхронизации почты и управления аутентификацией.

## Основные возможности

- Управление множеством ящиков: Эффективная работа с большим количеством одновременных IMAP-соединений.
- Безопасное хранение: Все локальные данные, включая учетные данные и заголовки писем, защищены шифрованием AES-256-GCM.
- Управление аутентификацией: Встроенный генератор TOTP (Time-based One-Time Password).
- Приоритет приватности: Хранение данных осуществляется только локально без использования сторонних облачных сервисов.
- Современный интерфейс: Отзывчивый стеклянный (glassmorphic) UI для десктопных систем.
- Системная интеграция: Нативные уведомления и фоновая синхронизация.

## Технологический стек

- Бэкенд: Go и фреймворк Wails.
- Фронтенд: HTML, Vanilla CSS, Javascript (Vite/Alpine.js).
- Безопасность: Шифрование AES-256-GCM.
- Протоколы: Стандартный IMAP через TLS.

## Требования

- Go 1.21+
- Node.js & NPM
- Wails CLI

## Разработка

Запуск в режиме разработки:
```bash
wails dev
```

Сборка приложения:
```bash
wails build
```
